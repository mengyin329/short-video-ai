# 核心模块Prompt模板

本目录存放4个核心模块的Prompt模板文件。

## 📁 文件命名规范

- `story_seed.txt` - 故事种子孵化器(StorySeed)
- `chara_weave.txt` - 角色关系编织师(CharaWeave)
- `rhythm_master.txt` - 叙事节拍控制师(RhythmMaster)
- `dialogue_forge.txt` - 对话铸造师(DialogueForge)

## 📝 如何添加你的Prompt

1. 在本目录下创建对应的`.txt`文件
2. 将你写好的Prompt内容直接粘贴到文件中
3. 保存文件即可

## ⚠️ 注意事项

- Prompt内容应该是纯文本格式
- 不需要添加特殊的标记或分隔符
- Agent会自动读取并使用这些Prompt
