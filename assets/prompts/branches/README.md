# 专业分支Prompt模板

本目录存放8个专业分支的Prompt模板文件。

## 📁 文件命名规范

- `ad_narrative_weave.txt` - 剧情类广告植入师(AdNarrativeWeave)
- `edu_story_weave.txt` - 知识科普编织师(EduStoryWeave)
- `healing_narrative.txt` - 情感治愈编剧师(HealingNarrative)
- `group_drama_director.txt` - 多人群戏导师(GroupDramaDirector)
- `romance_script_writer.txt` - 情侣关系编剧师(RomanceScriptWriter)
- `workplace_drama_writer.txt` - 职场关系编剧师(WorkplaceDramaWriter)
- `mystery_logic_master.txt` - 悬疑推理逻辑师(MysteryLogicMaster)
- `action_pace_director.txt` - 快节奏动作编剧师(ActionPaceDirector)

## 📝 如何添加你的Prompt

1. 在本目录下创建对应的`.txt`文件
2. 将你写好的Prompt内容直接粘贴到文件中
3. 保存文件即可

## ⚠️ 注意事项

- Prompt内容应该是纯文本格式
- 不需要添加特殊的标记或分隔符
- Agent会自动读取并使用这些Prompt
