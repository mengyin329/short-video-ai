# 短视频AI编剧助手 - 使用指南

## 🎬 项目概述

你已经拥有了一个专业的短视频AI编剧助手！这个Agent能够：

- ✅ **剧本创作**：完整剧本、故事大纲、角色设计、台词优化
- ✅ **分镜图生成**：单个/批量分镜图，自动生成宫格展示
- ✅ **场景设定图**：8种风格，支持上传参考图生成场景概念
- ✅ **角色设定图**：8种风格，支持演员照片参考、三视图生成

## 🚀 快速开始

### 方式一：直接描述需求（推荐）

直接在对话框中描述你的需求，Agent会智能理解并调用相应功能：

```
"帮我创作一个30秒的职场逆袭短视频"
"生成一个深夜便利店的场景设定图，日系清新风格"
"为这个角色生成设定图：25岁都市白领女性，韩系唯美风格"
"帮我把这个故事生成分镜图"
```

### 方式二：使用表单

1. 选择创作模式（完整流程/模块化/专业分支）
2. 填写创作需求
3. 点击「开始创作」

## 🎯 三种使用模式详解

### 模式一：完整流程模式

**触发方式：**
- 用户请求从零开始创作完整剧本
- 用户说"帮我创作一个短视频剧本"、"我要写一个剧本"等

**执行流程：**
```
1. 故事种子孵化器 → 收集创意，生成故事概念
       ↓
2. 角色关系编织师 → 设计角色，构建关系网络
       ↓
3. 叙事节拍控制师 → 规划节奏，设计场景时间表
       ↓
4. 对话铸造师 → 创作台词，完成完整剧本
```

**适用场景：**
- 需要从创意构思到完整剧本的全流程创作
- 不确定从哪里开始的新手创作者

---

### 模式二：模块化模式

**触发方式：**
- 用户明确需要某个特定功能
- 用户说"帮我设计角色"、"优化一下对话"等

**可用模块：**
| 用户请求示例 | 自动使用的模块 |
|-------------|---------------|
| "帮我设计一个角色" | 角色关系编织师 |
| "优化这段对话" | 对话铸造师 |
| "规划一下故事节奏" | 叙事节拍控制师 |
| "评估这个创意" | 故事种子孵化器 |

**适用场景：**
- 已经有了部分内容，只需要优化某个环节
- 明确知道自己需要什么功能

---

### 模式三：专业分支模式

**触发方式：**
- 用户明确需要某种特定类型的内容
- 用户说"我要写一个悬疑推理剧本"等

**可用分支：**
| 用户请求示例 | 自动使用的分支 |
|-------------|---------------|
| "写一个悬疑推理剧本" | 悬疑推理逻辑师 |
| "创作一个爱情故事" | 情侣关系编剧师 |
| "写一个广告植入的内容" | 剧情类广告植入师 |
| "做一个知识科普视频" | 知识科普编织师 |
| "写一个治愈系故事" | 情感治愈编剧师 |
| "设计一个多人场景" | 多人群戏导师 |
| "写一个职场故事" | 职场关系编剧师 |
| "设计一个动作场面" | 快节奏动作编剧师 |

**适用场景：**
- 明确知道内容类型
- 需要特定类型的专业化创作

## ⚙️ 高级配置

### 调整模型参数

编辑 `config/agent_llm_config.json`：

```json
{
  "config": {
    "model": "doubao-seed-1-8-251228",    // 模型选择
    "temperature": 0.85,                   // 创造性 (0-2，越高越有创意)
    "top_p": 0.95,                         // 采样参数
    "max_completion_tokens": 32768,        // 最大输出长度
    "timeout": 600,                        // 超时时间（秒）
    "thinking": "disabled"                 // 是否启用深度思考
  },
  "sp": "...",                             // 系统提示词（不建议修改）
  "tools": []
}
```

### 参数说明：

- **temperature**: 控制输出的创造性
  - 0.0-0.3：更确定性，适合需要精确内容的场景
  - 0.7-0.9：平衡创造性和准确性（推荐）
  - 1.0-2.0：高度创造性，适合创意写作

- **thinking**: 是否启用深度思考模式
  - "disabled"：快速响应（推荐）
  - "enabled"：深度推理，适合复杂逻辑任务

## 💡 使用技巧

### 1. 如何获得最佳效果？

**明确你的需求：**
- ✅ "我想写一个30秒的职场逆袭短视频，主题是新人被老员工欺负后反击"
- ❌ "帮我写个剧本"

**提供足够的信息：**
- 视频时长
- 目标受众
- 核心主题
- 想要的情感效果

### 2. 如何引导Agent进入特定模式？

**直接说明模式：**
- "请使用完整流程帮我创作"
- "我只需要你帮我设计角色"
- "请用悬疑推理逻辑师分支"

### 3. 如何迭代优化？

**分步骤优化：**
```
第一步："帮我创作一个故事概念"
第二步（在概念基础上）："帮我设计角色"
第三步（在角色基础上）："帮我规划节奏"
第四步（在节奏基础上）："帮我写台词"
```

## 🐛 常见问题

### Q1: Agent没有识别我的意图？

**解决方案：** 更明确地表达你的需求，比如：
- "我想用悬疑推理逻辑师分支创作一个剧本"
- "请用角色关系编织师帮我设计角色"

### Q2: 我的Prompt没有被使用？

**检查清单：**
1. 文件是否在正确的目录？
2. 文件名是否正确？
3. 文件是否保存成功？
4. 文件编码是否为UTF-8？

### Q3: 输出内容不符合预期？

**解决方案：**
1. 提供更详细的需求描述
2. 检查是否进入了正确的模式
3. 尝试调整temperature参数
